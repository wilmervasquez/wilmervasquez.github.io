Guía rápida para empezar con Git

Git es un sistema de control de versiones que te permite trabajar en equipo, llevar historial de tus proyectos y colaborar desde cualquier lugar 🌍.

Instalación

  1. Descarga Git desde git-scm.com.
  2. Verifica la instalación: 
    git --version
